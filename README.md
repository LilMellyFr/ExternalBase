## Overview

**External Base** is a foundational project for **external modification of Roblox (RBX) module bytecode**, intended **strictly for educational and research purposes**.

This project is designed to help experienced developers understand:

* RBX bytecode structures
* External memory interaction
* Module-level bytecode manipulation techniques

## Intended Audience

This project is **not beginner-friendly**.

You are expected to:

* Know how to **dump and maintain offsets**
* Understand **C++**, memory layouts, and reverse engineering basics
* Be capable of **maintaining and extending** the codebase yourself

If you’re planning to:

* Copy-paste without understanding
* Rebrand this as your own work
* Skip learning the underlying concepts

**This project is not for you.**
